# ИНН Парсер

Веб-приложение для массового получения ИНН физических лиц с сайта ФНС (Федеральной налоговой службы).

## Требования

- Python 3.10+
- Пакеты pip:
  - `pandas`
  - `playwright`
  - `starlette`
  - `uvicorn`
  - `sse-starlette`
  - `openpyxl`

## Установка

```bash
# Клонировать репозиторий
git clone <url> && cd prod

# Установить зависимости
pip install pandas playwright starlette uvicorn sse-starlette openpyxl

# Установить браузер Chromium для Playwright
playwright install chromium
```

## Запуск

```bash
python script/inn_web.py
```

Откройте в браузере: [http://localhost:1337](http://localhost:1337)

## Как это работает

1. **Загрузка файла** -- загрузите CSV или XLSX файл с данными физических лиц.
2. **Маппинг колонок** -- укажите, какие колонки содержат ФИО, дату рождения, серию и номер паспорта (и опционально дату выдачи документа).
3. **Автоматический парсинг** -- приложение открывает браузер Playwright, заполняет форму на сайте ФНС для каждой строки и перехватывает ИНН из сетевого ответа.
4. **Сохранение результата** -- результат сохраняется в CSV-файл с добавленной колонкой ИНН. Каждая строка сохраняется сразу после обработки, поэтому даже при прерывании частичные результаты не теряются.

## Поддерживаемые форматы файлов

- **CSV** -- разделитель `;`, поддерживаются кодировки UTF-8 и CP1251
- **XLSX** -- стандартный формат Excel

## Капча

Иногда сайт ФНС показывает капчу (CAPTCHA). В этом случае:
- Обработка приостанавливается
- В интерфейсе появляется уведомление
- Необходимо вручную решить капчу в окне браузера Playwright
- После решения нажмите кнопку "Капча решена, продолжить" в веб-интерфейсе
