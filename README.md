# ИНН Парсер

Веб-приложение для массового получения ИНН физических лиц с сайта ФНС (Федеральной налоговой службы).

## Быстрый старт (Windows)

1. Скачайте и установите [Python 3.10+](https://www.python.org/downloads/) (при установке поставьте галочку **"Add Python to PATH"**)
2. Скачайте этот репозиторий и распакуйте архив
3. Запустите `install.bat` — он установит все зависимости
4. Запустите `start.bat` — откроется сервер
5. Перейдите в браузере на [http://localhost:1337](http://localhost:1337)

## Ручная установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/flyokyla/inn-from-csv.git
cd inn-from-csv

# Установить зависимости
pip install -r requirements.txt

# Установить браузер Chromium для Playwright
playwright install chromium
```

### Запуск

```bash
python script/inn_web.py
```

Откройте в браузере: [http://localhost:1337](http://localhost:1337)

## Как это работает

1. **Загрузка файла** — загрузите CSV или XLSX файл с данными физических лиц
2. **Выбор колонок** — укажите колонки с ФИО, датой рождения, серией и номером паспорта. Опционально — дату выдачи документа. Выберите колонку для записи ИНН (существующую или новую). Задайте имя итогового файла
3. **Согласие на обработку данных** — при первом запуске в окне браузера Playwright появится форма согласия на обработку персональных данных. Подтвердите вручную (достаточно один раз — сохраняется в сессии браузера)
4. **Автоматический парсинг** — приложение заполняет форму на сайте ФНС для каждой строки и перехватывает ИНН из сетевого ответа
5. **Сохранение результата** — результат сохраняется в XLSX-файл после каждой строки — при прерывании частичные результаты не теряются

## Возможности

- Загрузка CSV и XLSX файлов
- Автоматическое определение колонок по типичным названиям
- Выбор существующей или создание новой колонки для записи ИНН
- Пользовательское имя итогового файла
- Прогресс в реальном времени (прогресс-бар, счётчики, лог)
- Кнопка «Закончить» — досрочная остановка с сохранением частичных результатов
- Уведомление о согласии на обработку персональных данных (при первом запуске)
- Уведомление и обработка капчи через веб-интерфейс

## Поддерживаемые форматы

- **CSV** — разделитель `;`, кодировки UTF-8 и CP1251
- **XLSX** — стандартный формат Excel

Результат всегда сохраняется в формате **XLSX**.

## Капча

Иногда сайт ФНС показывает капчу. В этом случае:
- Обработка приостанавливается
- В веб-интерфейсе появляется уведомление
- Решите капчу вручную в окне браузера Playwright
- Нажмите кнопку **"Капча решена, продолжить"** в веб-интерфейсе
